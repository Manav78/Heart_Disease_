# 📌 About the Project

This project was developed as a **college team assignment** to analyze **cardiovascular disease risk factors**.

We moved beyond static data by creating an **interactive ecosystem** that allows healthcare professionals and patients to visualize how **lifestyle choices** (smoking, alcohol, BMI) and **clinical factors** (diabetes, stroke history) correlate with heart disease risk.

The project combines **SQL preprocessing, Tableau visualization, and Flask web integration** to build a complete data analytics workflow.

---

# ⚙️ Technical Workflow

## 1️⃣ Data Acquisition & SQL Preprocessing

* **Storage:** Data was managed in a structured **SQL database**
* **Processing:** Used SQL for cleaning, handling missing values, and transforming raw features into analysis-ready formats
* **Calculated Fields:** Derived new metrics to enhance Tableau’s analytical depth

---

## 2️⃣ Tableau Visualization & Storytelling

* Developed **8–10 unique visualizations** including:

  * Heat Maps
  * Scatter Plots
  * Trend Charts

* Created a **Tableau Story Dashboard** to guide users through key insights and patterns in the dataset.

---

## 3️⃣ Web Integration (Flask)

* Developed a **Flask backend** to serve the application
* Integrated **Tableau dashboards** into a responsive **HTML5/CSS3 frontend**
* Enabled **real-time dashboard access through a web interface**

---

# 📊 Key Insights & Visualizations

### 👥 Demographic Analysis

Heart disease prevalence categorized by **Gender** and **Age groups**.

### 🚬 Lifestyle Impact

A deep dive into how **Smoking** and **Alcohol consumption** increase cardiovascular risk.

### 🏃 The "Activity Gap"

Visualization demonstrating how **regular physical activity helps reduce heart disease risk**.

### 🧠 Comorbidity Mapping

Overlap analysis of patients suffering from both **Diabetes** and **Stroke**.

# ▶️ How to Run

```

```

## 2️⃣ Launch Application

Run the main Python file:

```

```

## 3️⃣ Access Local Server

Open your browser and visit:

